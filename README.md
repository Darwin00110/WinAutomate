# WinAutomate

**WinAutomate** is a Windows automation tool developed in **Go**, designed to simplify repetitive system tasks quickly and efficiently. It is ideal for users who want to automate common processes such as managing programs, processes, and cleaning up temporary files.

---

## Features

- **Kill specific processes**  
  Example: `--killProcess --name <process_name>`

- **Open programs or folders**  
  Example: `--openProgram --path <path>`

- **Clean temporary files and cache**  
  Example: `--cleanCacheFiles --confirm`

> Note: Some functions may require administrator privileges.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com
```

2. Enter the directory:

cd WinAutomate

3. Run the program:

start main.exe
